 6. What are the versions of HTML?
Ans::-HTML 1.0 (1993):- This was the first official version of HTML, and it was a very basic markup language with limited elements and features. It laid the foundation for subsequent versions.

HTML 2.0 (1995): -HTML 2.0 introduced several new elements and features, such as forms and tables, which improved the ability to structure and format web content.

HTML 3.2 (1997): -HTML 3.2 was a significant step forward, with support for more styling options and the introduction of frames for layout.

HTML 4.01 (1999):- HTML 4.01 was a more refined version, with a strict and transitional doctype. It introduced better support for cascading style sheets (CSS) and improved form elements.

XHTML 1.0 (2000):- XHTML 1.0 was a reformulation of HTML as an XML application. It was designed to be more consistent and strict in its syntax, making it easier for developers to write clean, well-formed HTML.

HTML5 (2014):-HTML5 is the most recent major version of HTML. It introduced a wide range of new elements, attributes, and APIs, making it more powerful and versatile. HTML5 added native multimedia support (audio and video), semantic elements (like <header>, <nav>, and <article>), and a host of other features to enhance web development and multimedia integration.
