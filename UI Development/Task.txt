 6. What are the versions of HTML?
Ans::-HTML 1.0 (1993):- This was the first official version of HTML, and it was a very basic markup language with limited elements and features. It laid the foundation for subsequent versions.

HTML 2.0 (1995): -HTML 2.0 introduced several new elements and features, such as forms and tables, which improved the ability to structure and format web content.

HTML 3.2 (1997): -HTML 3.2 was a significant step forward, with support for more styling options and the introduction of frames for layout.

HTML 4.01 (1999):- HTML 4.01 was a more refined version, with a strict and transitional doctype. It introduced better support for cascading style sheets (CSS) and improved form elements.

XHTML 1.0 (2000):- XHTML 1.0 was a reformulation of HTML as an XML application. It was designed to be more consistent and strict in its syntax, making it easier for developers to write clean, well-formed HTML.

HTML5 (2014):-HTML5 is the most recent major version of HTML. It introduced a wide range of new elements, attributes, and APIs, making it more powerful and versatile. HTML5 added native multimedia support (audio and video), semantic elements (like <header>, <nav>, and <article>), and a host of other features to enhance web development and multimedia integration.

1.What is the use of the <table> tag in HTML?
Ans:-The <table> tag in HTML is used to create structured, tabular data on a web page. It is one of the fundamental HTML elements for displaying data in a grid format, typically consisting of rows and columns. The <table> tag is often used in combination with other elements like <tr> (table row), <th> (table header), and <td> (table data cell) to define the structure of the table.




 2. What is the use of the <tr> tag in HTML? 
Ans:-The <tr> tag in HTML is used to define a table row within an HTML table. It is an essential element for creating structured, tabular data on a web page. Each <tr> tag represents a single row within the table and contains one or more table cells (either header cells or data cells) defined using the <th> (table header) and <td> (table data cell) elements, respectively.


3. What is the use of the <th> tag in HTML?
Ans:-The <th> tag in HTML is used to define header cells within an HTML table. Header cells are typically used to label or describe the content of a column or row in a table. The <th> element provides semantic meaning to the content of a cell, indicating that it contains header information.


 4. What is the use of the <td> tag in HTML?
Ans:-The <td> tag in HTML is used to define data cells within an HTML table. Data cells are used to display the actual content or data in the rows and columns of a table. The <td> element is a fundamental part of creating structured, tabular data on a web page.


 5. What is the use of the <caption> tag in HTML? 
Ans:-The <caption> tag in HTML is used to add a title or caption to an HTML table. It provides a way to describe the purpose or content of the table, making it more accessible and understandable for users. The <caption> element is typically placed immediately after the opening <table> tag, and it is visually rendered above or below the table, depending on the CSS styling.


6. What is the use of the <ul> tag in HTML? 
Ans:-The <ul> tag in HTML is used to create an unordered list. An unordered list is a type of HTML list that presents items in a bulleted or unordered fashion. Each item in the list is typically preceded by a bullet point or some other marker, and the order of the items does not matter; they are not listed in a specific sequence.


7. What is the use of the <ol> tag in HTML?
Ans:-The <ol> tag in HTML is used to create an ordered list. An ordered list is a type of HTML list that presents items in a specific, sequential order, typically using numbers or other ordered markers to indicate the item's position in the list. Unlike unordered lists (created with the <ul> tag), the order of items in an ordered list is meaningful.


 8. What is the use of the <li> tag in HTML?
Ans:-The <li> tag in HTML is used to define list items within an ordered list (<ol>) or an unordered list (<ul>). It is a fundamental element for creating lists, whether they are ordered (numbered or lettered) or unordered (bulleted or customized markers). The <li> element represents individual items within the list and is used to define the content of each item.
